            I-Architecture UI:
1.Structure des dossiers :

public/ : Contient les fichiers statiques comme les images, les polices, etc.

src/ : Contient le code source de l'application.

assets/ : Contient les fichiers statiques tels que les images, les icônes, etc.

components/ : Contient les composants réutilisables.

pages/ : Contient les composants représentant les différentes pages de l'application.

services/ : Contient les services pour gérer les appels API et autres fonctions de communication avec le backend.

utils/ : Contient les fonctions utilitaires et les helpers.

hooks/ : Contient les hooks personnalisés.

context/ : Contient les contextes pour gérer l'état global avec React Context API.

styles/ : Contient les fichiers de styles globaux (CSS ou Sass).

App.jsx : Le composant principal de l'application.

main.jsx : Le point d'entrée de l'application.

vite.config.js : Le fichier de configuration de Vite.

package.json : Le fichier de configuration du projet.
